**MochiLang** is a Duolingo-style language learning app in progress—tiny lessons, satisfying taps, and a mascot who is extremely invested in your XP.

### What’s inside

- **Web app**: `apps/web` (Vite + React + TypeScript)
- **API**: `apps/api` (coming soon)

### Run it locally

From the repo root:

```bash
pnpm -C apps/web dev
```

Then open the URL Vite prints (usually `http://localhost:5173`).

### The vibe

- One-tap lessons on a little spiral path
- Popovers that pop like thought bubbles
- Buttons that look like you can *boop* them (because you can)
